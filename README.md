# AI Smart Waste Sorting System ♻️

## 📌 Project Overview
This project is an AI-based waste classification system that automatically classifies waste into categories such as plastic, paper, and metal using deep learning.

## 🚀 Features
- Image-based waste classification
- Deep learning model training
- Dataset preprocessing
- Model evaluation

## 🛠 Tech Stack
- Python
- TensorFlow / PyTorch
- OpenCV
- NumPy
- Streamlit (if used)
